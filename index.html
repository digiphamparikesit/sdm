Tentang Sumber Data

Dashboard ini mengambil data secara langsung dari Google Sheets melalui Apps Script Web App (bukan file statis), jadi setiap kali sheet data diperbarui, dashboard akan menampilkan angka terbaru setelah tombol "Muat Data" ditekan.

Belum terhubung
Cara pakai: buka Google Sheet → Ekstensi → Apps Script → pasang script backend → Deploy → Web app (Execute as: Me, Access: Anyone) → salin URL yang berakhiran /exec → tempel di kolom atas. Dashboard ini otomatis memanggil parameter ?type=sdm pada URL tersebut.
